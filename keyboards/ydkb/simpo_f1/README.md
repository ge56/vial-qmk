## Make

```
make ydkb/simpo_f1:xxx_vial
```

xxx means redux60rgb_ansi or other ...

## Convert BIN to UF2

```
uf2conv.py ydkb_simpo_f1_xxx_vial.bin -b 0x8004000 -c -f 0x9d5bcf1d -o xxx.uf2
```
