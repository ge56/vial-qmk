## Make

```
make ydkb/unicore_f1:xxx_vial
```

xxx means pavlov or hfkb ...

## Convert BIN to UF2

```
uf2conv.py ydkb_jolteon_mx_xxx_vial.bin -b 0x8004000 -c -f 0x9d5beef1-o xxx.uf2
```
